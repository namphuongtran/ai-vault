notes, references, and mental models, organized as a learning path from foundations to
advanced topics.

from the language switcher in the top navigation for native-language readers.

## Where to start

Begin with **[Foundations]({{< relref "foundations" >}})** — Stage 0. It covers the core
building blocks: foundation models, generative AI, prompt engineering, RAG, agents,
guardrails, model evaluation, and responsible AI.
